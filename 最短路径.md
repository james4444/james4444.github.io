# 数据结构与算法

## 最短路径

计算地图中的最短路线，我们把地图抽象成有向有权图，每个岔路口看作一个顶点，两个路口之间的路看作一条边，路的距离看作权重。这样计算最短距离，变成了计算有向有权图中计算两个顶点的最小距离。求解这个问题，有一个非常经典的算法，叫做**单元最短路径法**，最出名的就是Dijkstra算法了，算法分为两部分
* 构建有向有权图
```
public class Graph{
    private int v,
    private LinkedList<Edge> adj[];

    public Graph(){
        this.v = v;
        for(int i = 0; i < v; i++){
            adj[i] = new LinkedList<>();
        }
    }

    public void addEdge(int s, int t, int w){
        adj[s].add(new Edge(sid,tid,w))
    }

    private class Edge(){
        public int sid;
        public int tid;
        public int w;

        public Edge(int sid, int tid, int w){
            this.sid = sid;
            this.tid = tid;
            this.w = w;
        }
    }

    private class Vertex{
        public int id;
        public int dist; //记录从起点到顶点的距离
        public Vertex(int id, int dist){
            this.id = id;
            this.dist = dist;
        } 
    }

}
```
