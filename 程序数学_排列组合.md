# 程序数学

## 排列组合

### 排列：

**概念**

* 从n个不同的元素中取出m（1 <= m <= n）个元素,按照一定的顺序排成一列，这个过程就叫做排列。当n=m时，叫做全排列。
</br>

* 全排列的个数
**P(n,m) = n! / (n-m)!**
</br>
* 如果选择的元素是可重复的，这样的排列就是可重复排列，否则就是不可重复排列。

**应用**：暴力破解

## 组合

**概念**

* 从n个不同的元素中取出m(1 <= m <= n)个元素，不管其顺序合成一组，称为从n个元素中不重复的取出m个元素的一个组合。对于所有m取值的组合之全集，我们可以叫做全组合
</br>
* 组合的个数
**C(n,m) = n! / ((n-m)! * m!)**

**应用**：多元文法搜索
